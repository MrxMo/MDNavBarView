# MDNavBarView
下拉导航菜单（仿美团导航下拉菜单）
<br/>


## 支持
Android 4.0+
<br/>


## Gradle [下载aar](https://github.com/MrxMo/MDNavBarView/raw/master/release/mnavbarviewlib-v1.0.0.aar)

在项目的gradle中添加
```
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

在app的gradle中添加
 ```
 compile 'com.github.MrxMo:MPay:v1.0.1'
 ```
